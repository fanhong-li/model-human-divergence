# Model-Human Divergence in Video Physical Reasoning

Lightweight public repository for the ECCV 2026 paper:

> Human-Level Accuracy, Non-Human Strategies: Revealing Model-Human Divergence in Video Physical Reasoning

This repository is intentionally minimal while the full analysis code is being prepared for release after the camera-ready submission. It currently includes:

- `docs/training_process.md`: the training/evaluation protocol used for the frozen-encoder probes.
- `results_summary.csv`: a compact summary of the headline model-human results.

Full analysis and statistical scripts will be added only after an additional release review.

## Scope

Included:

- probe training protocol;
- model/backbone/evaluation configuration;
- aggregate result summary.

Not included yet:

- full analysis scripts;
- full per-sample predictions;
- benchmark video/HDF5 data;
- model checkpoints;
- statistical analysis code under review.
